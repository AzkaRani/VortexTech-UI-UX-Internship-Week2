# OZAAR – UX Case Study | Phase 2

## Overview

OZAAR is a mobile application that enables users to rent construction and home repair tools or hire verified technicians for household maintenance tasks. The platform offers doorstep delivery, secure identity verification, flexible payment options, and real-time order tracking to provide a convenient and reliable rental experience.

## Objective

The objective of this phase was to transform the low-fidelity wireframes into an interactive mid-fidelity prototype with improved layout, visual hierarchy, and user interactions.

- Mid-Fidelity Wireframes
